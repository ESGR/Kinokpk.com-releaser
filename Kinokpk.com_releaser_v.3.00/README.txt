Инструкция по установке/обновлению:
Загрузите файлы из папки upload на сервер, исключением папоки файлов: .settings, .bulidpath, .project
Загрузите файлы из папки to_forum_root в корневую директорию форума, если вы собираетесь использовать интерацию. Установите сначала форум!
Перейдите по адресу http://yoursite/install/ для установки и следуйте инструкциям установщика
Перейдите по адресу htpp://yoursite/update/ для обновления с версии 2.70 и следуйте инструкциям обновителя
